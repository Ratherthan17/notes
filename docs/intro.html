小工具​

计算机&&程序员​

  • 微软在线虚拟机

    • 备注：2小时后，重新打开，不行多试几次 (重新开启后的是新的，原先的东西会都没了，所以快到时间的时候记得备份资料)

  • Internet Speed Test（测网速的网站）

  • CrystalDiskInfo——检测硬盘固件、通电次数、通电时间等

    • 怀疑硬盘不是全新的？教你查看硬盘通电时间，以及SSD所剩寿命

  • AS SSD Benchmark——测试硬盘性能

  • 7-Zip——好用的压缩软件

  • Clash for Windows，Clash for Windows 汉化整合版——科学上网软件

  • Snipaste——好用的截屏软件，截的图可以悬浮在屏幕最上面

娱乐​

  • PotPlayer 中文网——好用的视频播放器

学习​

  • Quizlet——最佳的数字化单词卡和学习工具
  • MyGraphPaper 图形方格纸产生器——可以自定义纸张模板，比如语文本、英语本、田格、信纸、五线谱等
