步骤​

1. 下载系统镜像​

  • 点击 -> MSDN 官网——下载操作系统镜像（新版） <- 打开“我告诉你”官网，登陆进去，会弹出一个捐助窗口，可以捐助一下，也可以点击右下角的本次忽略。
  • 把鼠标放到 windows11 上，点击 详细信息 ，跳转页面后，往下划，找到 建议使用 ，选择一个系统镜像，点右边的复制，用百度网盘或迅雷下载到电脑上。

2. 制作启动盘​

  • 打开 FirPE 官网 ，点击 左上角 FirPE Project 左边的菜单栏（三个横线图标）， 应该默认在 目录 选项卡下，往下划，点击第⑧项 下载地址，然后选一个云盘下载
    • 推荐选择：官方指定免登陆不限速下载、东京蓝调的博客、IPFS CID（荷兰）下载分流，免登陆，不限速
    • 有一个需要注意的地方，如果选官方下载，不要点打包下载，而是点右上方的下载
  • 安装完后打开，选择要制作成启动盘的U盘，然后点击 全新制作
  • 制作完成后关闭，然后在U盘里新建个文件夹，取名叫系统，把下载的系统镜像文件复制进里面

3. 进入PE，安装系统​

3.1 进入PE​

  • 如果电脑本身系统就是 win10 或 win11 ，点击 开始 -> 电源 -> 按住 shift 键不放 点击 重启

    • 电脑会进入高级启动选项，点击 使用设备 -> EFI USB Device 按回车，电脑会重启进入PE系统

  • 如果本身系统不是 win10 或 win11 ，下载 图吧工具箱 ，打开后点击 主板工具 -> AIDA64

  • 打开 AIDA64 ，点击 计算机 -> 系统概述 ，在主板部分可以看到电脑的主板型号，然后在浏览器搜索该主板进入 bios 的按键
  • 搜到后重启，连续点按进入 bios 的按键，不要只按住不放，没用，进入 BIOS 后，怎么设置用U盘启动，上网搜下对应主板的设置吧，不同主板的 BIOS 界面有的地方不一样。

3.2 安装系统​

  • 如果成功进入PE了，找到之前放进U盘里的系统镜像，右键装载，会打开一个DVD或者CD驱动器的窗口，双击里面的　setup.exe ，开始安装系统

  • 安装部分，直接看我上一篇 Win10 的吧，我懒得写了

4. 安装时设置​

跳过联网​

备注
  • 如果这个方法失效了，可以用下这篇文章的方法2，文章：Windows 11如何跳过联网验证和绕过 Microsoft 帐户登录要求？
  • 等你完成添加完键盘布局等操作后，会发现 Win11 要求联网和登陆微软账户才能进行下一步
  • 按键盘上的 shift + F10 ，打开命令提示符，输入 oobe\bypassnro ，然后回车
  • 电脑会重启，然后自动进入 win11 的安装界面，等再次来到联网这一步，会发现 下一步 的左边多出一个 我没有internet连接
  • 点击它，接着选择 继续执行受限设置

5. 安装后设置​

5.1 系统激活​

  • 下载 云萌 Windows 10+ 激活工具 ，自动检测系统版本，一键激活
备注
  • 可能会显示连接不上微软的服务器，开下梯子再重新激活一下

5.2 右键菜单样式改回 win10​

  • 下载下面的批处理文件，一键切换 win10、win11 右键菜单样式
    切换windows菜单样式.bat

  • 输入 1 回车，会切换到 win10 右键菜单样式，输入 2 回车，会切换到 win11 右键菜单样式

5.3 添加桌面图标​

  • 对桌面右键 -> 个性化 -> 主题 -> 往下划，桌面图标设置 -> 勾选 计算机 和 控制面板 ，然后点击 应用

6. Win11 问题​

6.1 任务栏图栏消失，一片空白​

  • 点击 开始 -> 搜索 资讯 和 天气 ，将这两个应用卸载
